fix cfd/coupling/recurrence command

Syntax: 

fix ID group-ID couple/cfd/recurrence keyword values
  • ID, group-ID are documented in fix command
  • couple/cfd/recurrence = style name of this fix command
  • zero or more keyword/value pairs may be appended
  • keyword = transfer_density or transfer_type or transfer_force or transfer_fluctuations or transfer_tracer 
      transfer_* values = yes or no

Examples: 

fix		cfd2 all couple/cfd/recurrence transfer_force yes transfer_fluctuations yes

Description:

This fix exchanges the recurrence CFD relevant data between LIGGGHTS and CFDEMcoupling. Particle positions, velocities, radii and volumeweights are transferred from LIGGGHTS to CFDEMcoupling, the recurrence-prescribed velocities from CFDEMcoupling to LIGGGHTS. If needed, particle types and densities can also be communicated from LIGGGHTS to CFDEMcoupling (disabled per default) and forces (e.g. drag), velocity fluctuations and tracer concentrations from CFDEMcoupling to LIGGGHTS (disabled per default).

This fix needs a fix couple/cfd before it is defined.
